# Netflix Content Analytics Dashboard

## Project Overview
This project analyzes Netflix content data to understand movies, TV shows, genres, country-wise distribution, and content growth over time.

## Tools Used
- Python
- Pandas
- Power BI
- GitHub

## Dashboard Features
- Total Titles
- Total Movies
- Total TV Shows
- Total Countries
- Top 10 Genres
- Top 10 Countries
- Content Growth Over Years
- Movies vs TV Shows Distribution
- Type Slicer

## Key Insights
- Movies are more dominant than TV Shows.
- International Movies and Dramas are among the top genres.
- United States contributes the highest content.
- Netflix content increased rapidly after 2015.

## Files
- `netflix_analysis.ipynb` - Python data cleaning and EDA
- `cleaned_netflix_data.csv` - Cleaned dataset
- `netflix_dashboard.pbix` - Power BI dashboard
- `dashboard_screenshot.png` - Dashboard preview
